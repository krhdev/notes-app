# Note taking application

## Description
A note-taking app with persistent storage. Express backend serving a REST API + vanilla JS frontend, notes stored in a JSON file.


## Features
    - CRUD (Create, Read, Update, Delete)
    - Hosted on render
    - Notes have titles and body text
    - Timestamps showing when notes were created/updated
    - Persistent storage (notes survive a server restart)

## Installation
1. Clone the repo
```
git clone https://github.com/krhdev/notes-app.git
```

2. Install dependencies
```
npm install
```

3. Start the server
```
node server.js
```

4. Open your browser and visit
```
http://localhost:3001
```

note taking app - storing and saving important notes you can refer back to
